# Framework SuperClaude

SuperClaude es un framework de configuración de meta-programación que transforma Claude Code de un editor de código básico a una plataforma de desarrollo estructurada con agentes inteligentes y capacidades de automatización de flujos de trabajo.

## ¿Qué es SuperClaude?

SuperClaude mejora Claude Code inyectando instrucciones de comportamiento y orquestando componentes especializados. En lugar de trabajar solo con un editor de código, obtienes un sistema completo con:

- **Agentes Inteligentes**: 16 agentes especializados manejando expertise específico por dominio
- **Automatización de Flujos de Trabajo**: Gestión sistemática de tareas y orquestación
- **Optimización de Contexto**: Reducción dramática del uso de tokens (hasta 94% de reducción)
- **Modos de Comportamiento**: 7 modos diferentes que se adaptan al contexto de tu flujo de trabajo

## Componentes Principales

### Tres Plugins Principales

1. **PM Agent**
   - Orquestación impulsada por confianza
   - Gestión de tareas con umbral de confianza ≥90%
   - Toma de decisiones inteligente para enrutamiento de flujos de trabajo

2. **Research Plugin**
   - Capacidades de búsqueda web autónoma
   - Planificación adaptativa con razonamiento multi-salto
   - Recopilación de información consciente del contexto

3. **Index Plugin**
   - Motor de optimización de contexto
   - Logra 94% de reducción de tokens (58K → 3K tokens)
   - Mejora el rendimiento y la eficiencia de la ventana de contexto

### 16 Agentes Especializados

SuperClaude incluye agentes específicos por dominio para:
- Ingeniería de seguridad
- Arquitectura frontend
- Gestión de proyectos
- Desarrollo backend
- Automatización DevOps
- Y más...

Cada agente aporta conocimiento especializado y mejores prácticas a su dominio.

### 7 Modos de Comportamiento

El framework se adapta a diferentes contextos con modos especializados:

- **Brainstorming**: Ideación creativa y exploración
- **Business Analysis**: Análisis de requisitos y planificación estratégica
- **Deep Research**: Investigación exhaustiva y documentación
- **Orchestration**: Coordinación multi-agente
- **Token Efficiency**: Uso optimizado del contexto
- **Task Management**: Ejecución estructurada de flujos de trabajo
- **Introspection**: Auto-análisis y mejora

### 8 Servidores MCP

Capacidades mejoradas a través de servidores Model Context Protocol:
- **Tavily**: Búsqueda web avanzada
- **Serena**: Memoria y persistencia de contexto
- **Sequential**: Cadenas de razonamiento eficientes
- Más 5 servidores especializados adicionales

## Cómo Funciona

SuperClaude usa **plugins de TypeScript** con estas características clave:

- **Instalación Cero**: Auto-detección local del proyecto
- **Hot Reload**: Edita, guarda y ve cambios instantáneamente
- **Auto-Activación**: Se activa mediante el hook SessionStart
- **Código Base Maduro**: 298+ commits de desarrollo activo

## ¿Por Qué Usar SuperClaude?

### Eficiencia de la Ventana de Contexto

El Index Plugin puede reducir tu uso de tokens hasta en un 94%, permitiéndote:
- Trabajar con proyectos más grandes
- Mantener conversaciones más largas
- Mantener más contexto disponible
- Mejorar el rendimiento general

### Orquestación Inteligente

El PM Agent enruta inteligentemente tareas a agentes especializados basándose en:
- Complejidad y tipo de tarea
- Niveles de confianza (umbral ≥90%)
- Experiencia de dominio disponible
- Contexto actual del flujo de trabajo

### Automatización de Flujos de Trabajo

SuperClaude automatiza flujos de trabajo de desarrollo repetitivos:
- Investigación y documentación
- Análisis de seguridad
- Planificación de arquitectura
- Coordinación de tareas

SuperClaude es un framework construido por la comunidad y **no está afiliado ni respaldado por Anthropic**. Claude Code es un producto construido y mantenido por Anthropic.

## Casos de Uso

SuperClaude sobresale en:

- **Proyectos Complejos**: Bases de código grandes que requieren experiencia especializada
- **Tareas de Investigación Intensiva**: Proyectos que necesitan documentación y análisis extensivo
- **Trabajo Multi-Dominio**: Aplicaciones que abarcan seguridad, frontend, backend y DevOps
- **Entornos con Restricciones de Contexto**: Cuando la eficiencia de tokens es crítica
