# Content Templates

These templates provide structure and guidance for creating consistent, high-quality content for AIPaths Academy.

## Available Templates

- **[template-guide.md](./template-guide.md)** - For beginner guides and introductory content
- **[template-tutorial.md](./template-tutorial.md)** - For step-by-step project-based tutorials
- **[template-blog.md](./template-blog.md)** - For blog posts and articles

## How to Use

1. Copy the appropriate template
2. Fill in the frontmatter with your content details
3. Replace placeholder sections with your content
4. Follow the structure provided
5. Review against our [CONTRIBUTING.md](../CONTRIBUTING.md) guidelines

## Template Selection Guide

**Choose template-guide.md when:**
- Explaining a concept or technology
- Creating reference documentation
- Writing for beginners
- Goal: Understanding

**Choose template-tutorial.md when:**
- Building something step-by-step
- Teaching through practice
- Creating a complete project
- Goal: Building skills

**Choose template-blog.md when:**
- Sharing insights or opinions
- Announcing updates
- Writing thought leadership
- Goal: Engaging audience

## Tips for Great Content

✅ **Do:**
- Start with why it matters
- Use real-world examples
- Include code samples
- Test all code before publishing
- Provide next steps

❌ **Don't:**
- Assume too much knowledge
- Skip error handling in examples
- Forget to explain complex terms
- Leave broken links
- Ignore accessibility
